Go (sentry-go)

sentry.Init(sentry.ClientOptions{

})

Node.js (@sentry/node)

Sentry.init({
  dsn: "{{.DSN}}",
});

Python (sentry-sdk)

sentry_sdk.init(
    dsn="{{.DSN}}",
)

Note: sentry-sdk requires a numeric project ID in the DSN, and gzip-compresses envelopes by default (Trackdown handles this automatically).
